# GCP_Infra_Project

Project ideas:
-  Zautomatyzowane wdrażanie aplikacji w Golangu na GKE z konfiguracją sieciową i bezpieczeństwem za pomocą Terraform
-  Bezpieczna komunikacja między aplikacjami w Golangu w chmurze z wykorzystaniem VPN i Google Cloud Interconnect
- Zabezpieczony dostęp do aplikacji w Golangu za pomocą Identity-Aware Proxy i load balancing w Google Cloud
- Zarządzanie dynamiczną siecią i dostępem do aplikacji w Golangu w Google Cloud z wykorzystaniem Terraform, VPC i Cloud NAT
- Wysoka dostępność aplikacji w Golangu z replikacją w różnych strefach dostępności w Google Cloud
- Zarządzanie dostępem i monitorowanie aplikacji w Golangu z wykorzystaniem Google Cloud Logging, Monitoring i VPC Flow Logs
- Konfiguracja rozproszonego systemu kolejkowego w Golangu z użyciem Google Cloud Pub/Sub oraz zabezpieczenie komunikacji przez VPC
- Zabezpieczenie aplikacji w Golangu przed atakami DDoS za pomocą Google Cloud Armor oraz automatyczne wdrożenia z Terraform
- Zarządzanie dostępem do aplikacji w Golangu z użyciem VPC Service Controls i Identity-Aware Proxy
